1. 두 개의 정수 n과 m이 주어집니다. 반복문을 사용하지 않고 별(*) 문자를 이용해 가로 의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요

   ```python
   # 문제
   n = 5
   m = 9
   정답코드
   n = int(input("가로의 길이를 입력하세요."))
   m = int(input("세로의 길이를 입력하세요."))
   rec = m * (n * '*' + '\n')
   print(rec)
   ```
